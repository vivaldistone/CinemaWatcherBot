CinemaWatcherBot

CinemaWatcherBot — это backend-приложение, которое автоматически собирает информацию о киносеансах с сайтов кинотеатров и предоставляет её через Telegram-бота.

Проект использует Playwright для парсинга динамического контента и Background Services для периодического обновления данных.

Технологии:
ASP.NET Core
Playwright
Telegram Bot API
Background Services
Dependency Injection
Clean Architecture
